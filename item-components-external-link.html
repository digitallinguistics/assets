5.11 External Link

A link to an external website. Apply this style using a pattern similar to: a:not([href*='danielhieber.com'])

Example

This is a paragraph with an inline link. This sentence contains a link to an external website.

<p>This is a paragraph with an <a class=link href=#>inline link</a>. This sentence contains a <a href=https://danielhieber.com class='link external-link'>link to an external website</a>.</p>
